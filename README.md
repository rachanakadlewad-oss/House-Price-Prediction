# 🏠 House Price Prediction

A Machine Learning web application built with **Streamlit** that predicts house prices in **Bangalore** based on user inputs such as location, total square feet, number of bathrooms, and BHK. The model is trained using the **Random Forest Regressor** algorithm.

---

## 📌 Features

- Predicts house prices based on user inputs
- Interactive and user-friendly Streamlit interface
- Data preprocessing and feature engineering
- Exploratory Data Analysis (EDA)
- Machine Learning model trained using Random Forest Regression
- Fast and accurate predictions

---

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/your-username/house-price-prediction.git
```

### Move into the project directory

```bash
cd house-price-prediction
```

### Create a virtual environment (Optional)

```bash
python -m venv venv
```

Activate it

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

The application will open in your browser at:

```
http://localhost:8501
```

## 📈 Dataset

The project uses the **Bengaluru House Price Dataset**, containing information such as:

- Location
- Total Square Feet
- BHK
- Bathrooms
- Price

The dataset is cleaned and preprocessed before training the model.
